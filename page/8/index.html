Android应用性能优化 ch4.1

2017-08-08 技术

说说内存

Android与传统JVM差别：

  • 大小
  • 交换能力
    • 内存交换能力——给应用系统有更多内存可用的假象
    • Android根本没有PC机上虚拟内存大内存寻址空间的优待

primary type内存占用

Type Local type Size
boolean jboolean 8bit(desired by VM)
byte jbyte 8
char jchar 16
short jshort 16
int jint 32
long jlong 64
float jfloat 32
double jdouble 64

使用尽可能少的内存，既是经验，也是常识

  • 除了减少OOM的风险外，也可提升性能

性能主要由以下三个因素决定：

  • CPU如何操纵特定的数据类型
  • 数据和指令需要占用多少存储空间
  • 数据在内存中的布局
  • Android
  • Performance

【国学经典】《论语》-学而第一-13

有子曰：信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也

信近于义，言可复也
——“言必行，行必果”，但前提条件是讲信用也要符合道义，说出来的话也需要是遵从礼仪的去做才合适

恭近于礼，远耻辱也
——平时毕恭毕敬所侍之人仁义、所做之事合道，才能够与耻辱之人、之事理清瓜葛

因不失其亲，亦可宗也
——所依赖的人都是信、义、礼之辈，那这个也是可以依赖的

引申：
系孔门弟子 有子 所言，凸显他们对礼、义之重视
最后的点睛之句亦类似“亲君子远小人”
几位弟子对老师教义的领会是比较到位的，不过知易行难，实践中是哪般定是难说的

【国学经典】《论语》-学而第一-15

2017-08-08 论语

子贡曰：“贫而无谄，富而无骄。何如？”子曰：“可也。未若贫而乐，富而好礼者也。”子贡曰：“《诗》云：如切如磋，如琢如磨。其斯之谓与？”子曰：“赐也，始可与言《诗》已矣。告诸往而知来者。”

子贡，孔子学生，名赐

贫而无谄，富而无骄。何如？
——贫穷的时候不巴结别人，富贵的时候不骄傲，怎么样？

可也，未若贫而乐，富而好礼者也
——好是好，但是不如，贫穷的时候保持快乐，富贵的时候尊崇礼

《诗》云：如切如磋，如琢如磨。其斯之谓与？
——《诗经》有云，如切如磋，如琢如磨。讲的就是这个道理吧？

赐也，始可与言《诗》已矣。告诸往而知来者。
——赐啊，你能够举一反三，我可以跟你谈论谈论《诗经》了

引申：
常人，能够想到“贫而无谄，富而无骄”的是少数；
子贡问老师，老师指点之后，子贡马上能举一反三，可以看出子贡的悟性，他也得到了夫子的赞扬。

“贫而乐，富而好礼”是夫子所倡导的态度；能够举一反三，也是夫子最期望弟子们习得的能力。

【论语&孙子兵法】Day-18

2017-08-08 论语孙子兵法

【论语&孙子兵法】Day 18

原文与注解

子曰：“诗三百首，一言以蔽之，曰，思无邪”

《诗经》三百篇，一句话概括，思想纯正；三百，约指

体会

这大概可以解释为什么有如此多的人给子孙后代起名的时候，喜欢翻阅诗经

原文与注解

其用战也胜，久则钝兵挫锐，攻城则力屈，久暴师则国用不足。夫钝兵挫锐、屈力殚货，则诸侯乘其弊而起，虽有智者，不能善其后矣。故兵闻拙速，未睹巧之久也。夫兵久而国利者，未之有也。故不尽知用兵之害者，则不能尽知用兵之利也。

讲就用兵之害、用兵之利

体会

兵贵神速，关键时刻，不能给敌人喘息的机会，战场上的关键时机，用“一招失，满盘输”形容不为夸张
