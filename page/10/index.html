Android应用性能优化 ch4.1

2017-08-08 技术

说说内存

Android与传统JVM差别：

  • 大小
  • 交换能力
    • 内存交换能力——给应用系统有更多内存可用的假象
    • Android根本没有PC机上虚拟内存大内存寻址空间的优待

primary type内存占用

Type Local type Size
boolean jboolean 8bit(desired by VM)
byte jbyte 8
char jchar 16
short jshort 16
int jint 32
long jlong 64
float jfloat 32
double jdouble 64

使用尽可能少的内存，既是经验，也是常识

  • 除了减少OOM的风险外，也可提升性能

性能主要由以下三个因素决定：

  • CPU如何操纵特定的数据类型
  • 数据和指令需要占用多少存储空间
  • 数据在内存中的布局
  • Android
  • Performance

原文与注解

子曰：“不患人之不己知，患不知人也。”

不怕别人不了解、理解、领悟自己的意思、德行，怕不理解别人、不懂别人啊！

体会

一己之私，真的那么重要么？

你能给别人带来何样的价值，决定了下一步得到的反馈，若真是不被理解，又由何妨呢？

真理自在

原文与注解

夫未战而庙算胜者，得算多也；未战而庙算不胜者，得算少也。多算胜，少算不胜，而况于无算乎！吾以此观之，胜负见矣。

未战而能预料到胜利，因为筹备周密、准备充分；不能预料胜利的，准备不足也。准备的充分与否，关乎胜负，这是显而易见的。

体会

其用战也胜，久则钝兵挫锐，攻城则力屈，久暴师则国用不足。夫钝兵挫锐、屈力殚货，则诸侯乘其弊而起，虽有智者，不能善其后矣。故兵闻拙速，未睹巧之久也。夫兵久而国利者，未之有也。故不尽知用兵之害者，则不能尽知用兵之利也。
