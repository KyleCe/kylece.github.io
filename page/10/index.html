原文与注解

夫未战而庙算胜者，得算多也；未战而庙算不胜者，得算少也。多算胜，少算不胜，而况于无算乎！吾以此观之，胜负见矣。

未战而能预料到胜利，因为筹备周密、准备充分；不能预料胜利的，准备不足也。准备的充分与否，关乎胜负，这是显而易见的。

【国学经典】《论语》-学而第一-14

2017-08-08 论语

子曰：君子食无求饱，居无求安。敏于事而慎于言，就有道而正焉。可谓好学也已。

君子食无求饱，居无求安
—— 食不求珍馐，居不求豪宅

敏于事而慎于言
—— 做事机敏，说话谨慎（所谓谨言慎行）

就有道而正焉
—— 参考德至于道这一境界的人来匡正自己的言行

这样才算是好学。

引申：
将精力聚焦在有价值的事情上，其它的细节你真的需要考虑这么多吗？别样的物质，你真的需要拥有那么多吗？

Android应用性能优化 ch4.1

2017-08-08 技术

说说内存

Android与传统JVM差别：

  • 大小
  • 交换能力
    • 内存交换能力——给应用系统有更多内存可用的假象
    • Android根本没有PC机上虚拟内存大内存寻址空间的优待

primary type内存占用

Type Local type Size
boolean jboolean 8bit(desired by VM)
byte jbyte 8
char jchar 16
short jshort 16
int jint 32
long jlong 64
float jfloat 32
double jdouble 64

使用尽可能少的内存，既是经验，也是常识

  • 除了减少OOM的风险外，也可提升性能

性能主要由以下三个因素决定：

  • CPU如何操纵特定的数据类型
  • 数据和指令需要占用多少存储空间
  • 数据在内存中的布局
  • Android
  • Performance

【国学经典】《论语》-学而第一-10

2017-08-08 论语

子禽问于子贡曰：“夫子至于是邦也，必闻其政。求之与？抑与之与？”子贡曰：“夫子温良恭俭让以得之。夫子求之也，其诸异乎人之求之与？”

夫子
敬称，这里指代孔老夫子

求之与，抑与之与
抑：还是。这里是对孔子不管所到何国，必受问政这一现象的疑惑。到底是诸侯求问于夫子呢？还是夫子自己争取的？

夫子温良恭俭让 以得之
夫子，温良恭俭让，才得如此款待，即使是夫子自己争取的机会，也不同于一般唯名利是图的人吧？

主要是对夫子的赞誉，纵名副其实

2017-08-07 随笔

坚持&改进&反省

才写了几天，就有点不想写了，足以见得坚持是一件多么难的事情。

”你想要得到的，最好的办法就是让自己配得上。“——查理芒格

今年以来，自我更新的频率明显变慢了，上半年找到的借口是“在创业公司，一周六天班”，换完工作，似乎又找到了新的借口”这儿的工作比创业公司占用的时间还要多“。

  • 不要寻找借口，不要抱怨，不要再说”没有时间“
  • 要保持学习，保持进步，读好书，思考改进。

工作上

总是在说新来这地方，想法得不到认可采纳，可实际上，你做的还不够。

  • 性能优化的具体实践上，你并没有找到关键问题点，虽然总在超额完成被分配的任务，但做的还不够好，没有解决问题，永远都是没有解决问题

  • 脚本，虽然上上周末做到80%，但不是100%永远也不是100%；

    况且，你有没有想过，需要你把这件事情做到100%么？这样把其它同事的事情做到他做不到的程度，真的利于一个team的团结向上么？

  • 工作报告、分享上，有落入形式主义陷阱的趋势，需要悬崖勒马

    重要的永远是内容，画龙点睛之笔应放在最后的最后

学习上

多久没看完过一本技术书了？多久没好好学英语了？多久没有把年初的计划拿来执行了？

——这样看来，离心智成熟还差的有一段距离啊
