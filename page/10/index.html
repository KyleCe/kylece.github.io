其用战也胜，久则钝兵挫锐，攻城则力屈，久暴师则国用不足。夫钝兵挫锐、屈力殚货，则诸侯乘其弊而起，虽有智者，不能善其后矣。故兵闻拙速，未睹巧之久也。夫兵久而国利者，未之有也。故不尽知用兵之害者，则不能尽知用兵之利也。

【国学经典】《论语》-学而第一-13

2017-08-08 论语

有子曰：信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也

信近于义，言可复也
——“言必行，行必果”，但前提条件是讲信用也要符合道义，说出来的话也需要是遵从礼仪的去做才合适

恭近于礼，远耻辱也
——平时毕恭毕敬所侍之人仁义、所做之事合道，才能够与耻辱之人、之事理清瓜葛

因不失其亲，亦可宗也
——所依赖的人都是信、义、礼之辈，那这个也是可以依赖的

引申：
系孔门弟子 有子 所言，凸显他们对礼、义之重视
最后的点睛之句亦类似“亲君子远小人”
几位弟子对老师教义的领会是比较到位的，不过知易行难，实践中是哪般定是难说的

Android应用性能优化 ch4.1

2017-08-08 技术

说说内存

Android与传统JVM差别：

  • 大小
  • 交换能力
    • 内存交换能力——给应用系统有更多内存可用的假象
    • Android根本没有PC机上虚拟内存大内存寻址空间的优待

primary type内存占用

Type Local type Size
boolean jboolean 8bit(desired by VM)
byte jbyte 8
char jchar 16
short jshort 16
int jint 32
long jlong 64
float jfloat 32
double jdouble 64

使用尽可能少的内存，既是经验，也是常识

  • 除了减少OOM的风险外，也可提升性能

性能主要由以下三个因素决定：

  • CPU如何操纵特定的数据类型
  • 数据和指令需要占用多少存储空间
  • 数据在内存中的布局

【国学经典】《论语》-学而第一-11

2017-08-08 论语

子曰：父在，观其志。父没，观其行。三年无改于父之道，可谓孝矣。

其
——他的，非父

没
——逝

父之道
——道，可谓德之道

三年无改 可谓孝
——能够较长时间坚持父之道，可以算作是孝道

引申：
字面意思理解，或许会感觉比较浅薄、迂腐；
联系“三人行必有我师焉，择其善者而从之，其不善者而改之”，似乎此处的“三年无改”略显僵化，实则涉及到两个方面：

  1. 道的界限把握；
  2. 虽不适，实行之；强调的是对待值得尊敬的人时的度量、牺牲、隐忍…纵其没

坚持&改进&反省

才写了几天，就有点不想写了，足以见得坚持是一件多么难的事情。

”你想要得到的，最好的办法就是让自己配得上。“——查理芒格

今年以来，自我更新的频率明显变慢了，上半年找到的借口是“在创业公司，一周六天班”，换完工作，似乎又找到了新的借口”这儿的工作比创业公司占用的时间还要多“。

  • 不要寻找借口，不要抱怨，不要再说”没有时间“
  • 要保持学习，保持进步，读好书，思考改进。

工作上

总是在说新来这地方，想法得不到认可采纳，可实际上，你做的还不够。

  • 性能优化的具体实践上，你并没有找到关键问题点，虽然总在超额完成被分配的任务，但做的还不够好，没有解决问题，永远都是没有解决问题

  • 脚本，虽然上上周末做到80%，但不是100%永远也不是100%；

    况且，你有没有想过，需要你把这件事情做到100%么？这样把其它同事的事情做到他做不到的程度，真的利于一个team的团结向上么？

  • 工作报告、分享上，有落入形式主义陷阱的趋势，需要悬崖勒马

    重要的永远是内容，画龙点睛之笔应放在最后的最后

学习上

多久没看完过一本技术书了？多久没好好学英语了？多久没有把年初的计划拿来执行了？
